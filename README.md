
# AI Resume Builder

* I have built Builder. IO to make resume building simple, fast, and ATS-friendly.
* Live at : https://lnkd.in/gsn6XhKp


 # Core Features
 - Live Preview while editing
 - Enhance descriptions using AI
 - ATS-friendly, industry-standard resumes
 - Multiple professional templates
 - Edit your resumes anytime

# Tech Stack Used -
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- LLM (Enhance feature): gpt-oss-20b (Groq)
- Deployment: Vercel & Render

This project has given me a better understanding of system design, react rendering, handling database.
